# Chrome Extensions

Este repositorio contiene una colección de extensiones para Google Chrome desarrolladas por Baxahaun.

Cada extensión reside en su propia carpeta y tiene su propio código fuente, documentación y licencia específica.

## Extensiones Incluidas

*   **[Screenshot Pro](./Screenshot-extension):** Una potente extensión para capturar la pantalla en alta calidad (WebP) que incluye algoritmos para evadir técnicas anti-screenshot (overlays superpuestos, bloqueos de clic, etc).

## Estructura del Repositorio

La estructura general del repositorio es la siguiente:
```
.
├── Nombre-Extension-1/
│   ├── manifest.json
│   ├── background.js
│   ├── content.js
│   ├── README.md
│   └── LICENSE
├── Nombre-Extension-2/
│   ├── manifest.json
│   ├── ...
│   └── LICENSE
└── README.md
```

## Licencias

Las licencias pueden variar dependiendo de la extensión. Por favor, consulta el archivo `LICENSE` dentro del directorio de cada extensión para ver los detalles específicos.
